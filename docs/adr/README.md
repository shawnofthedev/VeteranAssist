# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for VeteranEvidenceAssist.

ADRs are short records of decisions that affect architecture, privacy, security, data handling, or product boundaries. They are intentionally concise so future contributors can understand why the project is shaped the way it is.

## Status Values

- Proposed: Under consideration.
- Accepted: Current project direction.
- Superseded: Replaced by a later ADR.
- Deprecated: No longer recommended, but not yet replaced.
